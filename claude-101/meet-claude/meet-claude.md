# Meet Claude
Before chatting with Claude, set the following for effective prompts:
Setting the stage:
  * Role and objective
  * Context that about the work that Claude should know about
Define the task:
  * What actions for Claude to do
Specifying rules:
  * Style or tone for Claude? Provide examples for Claude: what am I looking at?
  
Example:
I'm the marketing lead at an indie streaming startup, and we're preparing an investor pitch deck. (1)
Can you research the current state of the independent film streaming market and identify key trends, competitor positioning, and growth opportunities? (2)
Use current web research with citations and structure it as a professional report.(3)

Adding context：
  * Claude can analyze files (images, pdf, etc)
  
Iterating on Claude's responses:
  * Ask follow-up questions:Build on Claude's response, ask more. 
  * Provide feedback:Let Claude knows the response (good? bad?)
  * Redirect or restart: Steer it back if Claude is off, or restart.

Getting better results:
  * Claude's response is too generic: Add more context
  * The response is too lengthy (or too short): Be explicit about the length 
  * Claude didn't follow my format: Provide examples to Claude 
  * Claude is making up:  Ask Claude to cite sources or indicate confidence level. Enable web search to ground responses in current information.
  * Tone is wrong: Describe the tone in plain language.
  
The key is to iterate: Treat the first response as a starting point. Review -> Identify what is working, what is not -> refine 
Provide specific feedback: Instead of "make it shorter", "Cut the first two paragraphs and make the conclusion more action-oriented"
Know when to start fresh: If the conversation is off the track, restart.

AI Fluency:
The ability to collaborate effectively with AI tools. Developing the judgment to use AI well across different situations.
* Deligation: Deciding on what work should be done by human and what should be done by AI, and distributing the tasks. Understand your goals, AI capabilities and making strategic choices
* Description: Effectively communicating with AI systems. Clearly defining outputs, guiding AI processes, and specifying desired AI behaviors and interactions.
* Discernment: Critcally evaluating AI outputs. Assessing quality, accuracy, appropriateness, and determining areas for improvement
* Diligence: Using AI responsibly and ethically.

Evaluate Claude:
Is it good at the task?
Why: 
* Understand where Claude adds the most value in your workflow
* Identify tasks where you'll need to provide more context or examples
* Build confidence in Claude's outputs for recurring tasks

Example approach for eval:
1. Gathter 5-10 examples
2. Create test prompt. Write prompts that would generate similar outputs. 
3. Compare outputs. Run the prompts and compare to Claude's responses: Does Claude capture the key information? Is it appropriate? What could be improved? 
4. Refined the approach. Adjust the prompts, add examples, etc, based on the finding.
